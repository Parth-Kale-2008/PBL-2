# 🥗 AI Packaged Food Analyzer

> Scan smart. Eat smart.

An AI-powered web app that helps users check whether packaged food products are healthy or not by simply entering the barcode number.

The app fetches real product data, ingredients, and allergens, then uses AI to generate a health score and verdict.

# 🚀 Features

✅ Enter barcode number to analyze products  
✅ Health Score out of 10  
✅ Final Verdict (Healthy / Moderate / Avoid)  
✅ Detect allergens like soy, gluten, milk, nuts, etc.  
✅ Ingredient-based AI analysis  
✅ Clean and simple UI  

# 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  

### APIs Used

- **Open Food Facts API** → Fetches product details, ingredients, allergens  
- **OpenRouter API** → AI health analysis, score, verdict  

# ⚙️ How It Works

Enter Barcode

     ↓

     ↓
